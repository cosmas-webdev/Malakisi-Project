# Malakisi-Project
Malakisi-Project est une réalisation pratique développée avec le Programme Malakisi de DRC MIND, où j'ai appliqué mes compétences en HTML5 et Tailwind CSS pour créer une interface web moderne et responsive. Ce projet incarne l'approche "learning by doing" et témoigne de ma montée en compétences grâce à cet encadrement de qualité.
