## 📋 **Description**

**Malakisi-Project** est une réalisation pratique développée dans le cadre du prestigieux **Programme Malakisi** initié par **DRC MIND** en partenariat avec l'**Ambassade des États-Unis en RDC**. 

Ce projet démontre la maîtrise des compétences suivantes :
- 🎨 Création d'interfaces web modernes et professionnelles
- 📱 Design responsive avec approche **mobile-first**
- 🧩 Utilisation avancée de **Tailwind CSS**
- ⚡ Performance et optimisation du code frontend

---

## ✨ **Caractéristiques du Projet**

### 🎯 **Objectifs Pédagogiques**
- Appliquer les principes du design responsive
- Maîtriser les classes utilitaires de Tailwind CSS
- Créer une interface utilisateur intuitive
- Optimiser l'expérience utilisateur (UX/UI)

### 🎨 **Éléments de Design**
- ✅ **Layout responsive** (mobile, tablette, desktop)
- ✅ **Composants réutilisables** (cartes, boutons, navigation)
- ✅ **Grille flexible** avec Tailwind CSS Grid
- ✅ **Typographie soignée** et hiérarchie visuelle
- ✅ **Palette de couleurs** cohérente et professionnelle
- ✅ **Animations subtiles** pour l'interaction utilisateur
- ✅ **Formulaire de contact** stylisé
- ✅ **Navigation intuitive** avec menu responsive

### 📱 **Responsive Design**
| Appareil | Breakpoint | Adaptation |
|----------|------------|------------|
| Mobile | < 640px | Menu hamburger, colonnes empilées |
| Tablette | 640px - 1024px | Grille 2 colonnes |
| Desktop | > 1024px | Layout complet avec sidebar |
